# Song Collection Manager 🎵

A Java console application to manage a personal music collection 
with albums and songs.

## Features
- Create and manage up to 4 albums (max 5 songs each)
- Add, delete, and search songs
- Filter songs by genre (rock, pop, hip-hop, bossa nova)
- Filter songs by duration
- Load albums and songs from a text file

## How to Run
1. Make sure Java is installed on your machine
2. Clone the repo
3. Compile all files:
```
javac Song.java Album.java SongCollection.java
```
4. Run the program:
```
java SongCollection
```

## Project Structure
- `Song.java` — Song class with name, artist, duration, genre
- `Album.java` — Album class managing up to 5 songs
- `SongCollection.java` — Main class with menu-driven interface
- `ReginaCollection.txt` — Sample data file
